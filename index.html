Applied Data Science Lab at WQU

Module 1 – Housing in Mexico: Use a dataset of 21,000 properties to determine if real estate prices are influenced more by property size or location; Import and clean data from a CSV file, build data visualizations, and determine the relationship between two variables using correlation.

Module 2 – Apartment Sales in Buenos Aires: Build a linear regression model to predict apartment prices in Argentina; Create a data pipeline to impute missing values and encode categorical features; Improve model performance by reducing overfitting.

Module 3 – Air Quality in Nairobi: Extract data from a MongoDB Database using pymongo; Build an ARMA time-series model to predict particulate matter levels in Kenya; Improve model performance through hyperparameter tuning.

Module 4 – Earthquake Damage in Nepal: Extract data from an SQLite database; Build logistic regression and decision tree models to predict earthquake damage to buildings; and reveal the biases in data that can lead to discrimination.

Module 5 – Bankruptcy in Poland: Build random forest and gradient boosting models to predict whether a company will go bankrupt; Navigate the Linux command line; Address imbalanced data through resampling and consider the impact of performance metrics precision and recall.

Module 6 – Customer Segmentation in the US: Build a k-means model to cluster US consumers into groups; Use principal component analysis (PCA) for data visualization; and create an interactive dashboard with Plotly Dash.

Module 7 – A/B Testing at WorldQuant University: Conduct a chi-square test to determine if sending an email can increase program enrollment at WQU; Build custom Python classes to implement an ETL process, and create an interactive data application following a three-tiered design pattern.

Module 8 – Volatility Forecasting in India: Create a GARCH time-series model to predict asset volatility; Acquire stock data through an API; Clean and store it in an SQLite database; Build own API to serve model predictions..
